# 红黑树是怎么实现平衡的？它的优点是什么？

平衡有三种方式：

- 变色
- 左旋
- 右旋

变色：黑色变为红色，红色变为黑色

左旋：

```
    \                                  \
     X                      \           Y
    / \            ==========\         / \
   a   Y           ==========/        X   c
      / \                   /        / \
     b   c                          a   b
```

右旋：

```
    \                                  \
     X                      \           Y
    / \            ==========\         / \
   Y   a           ==========/        b   X
  / \                       /            / \
 b   c                                  c   a
```
