```

输入: 1->2->3->4->5->NULL
输出: 5->4->3->2->1->NULL
```

1. 迭代

   ```go
   func reverseList(head *ListNode) *ListNode {
   	var prev *ListNode
   	for head != nil {
   		head.Next, head, prev = prev, head.Next, head
   	}
   	return prev
   }
   ```

   记：三个指针，head下一个变前一个（转变方向），head往后一个，prev往后一个
